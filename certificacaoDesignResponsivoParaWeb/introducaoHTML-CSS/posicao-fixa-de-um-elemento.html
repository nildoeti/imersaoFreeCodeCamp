Posicionamento fixo de um elemento

O próximo esquema de layout que o CSS oferece é o fixedposition, que é um tipo de posicionamento absoluto que bloqueia um elemento em relação à janela do navegador. Semelhante ao posicionamento absoluto, é usado com as propriedades de deslocamento CSS e também remove o elemento do fluxo normal do documento. Outros itens não "percebem" mais onde estão posicionados, o que pode exigir alguns ajustes de layout em outro lugar.

Uma diferença fundamental entre o fixede absoluteposições é que um elemento com uma posição fixa não se moverá quando o usuário rolar.

A barra de navegação no código é rotulada com uma class de navbar. Mudar o seu positionpara fixed, e desloque-o em 0 pixels do tope 0 pixels do left. Depois de adicionar o código, role a janela de visualização para ver como a navegação permanece no lugar.
